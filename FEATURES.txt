13. TECHNICAL STACK
================================================================================

13.1 Frontend
     - HTML5 + Vanilla JavaScript (SPA)
     - TailwindCSS (via CDN)
     - Custom CSS (styles.css)
     - Material Symbols (Google Icons)
     - Public Sans font (Google Fonts)
     - Hash-based routing (Router.js)
     - Component-based page renderers (Pages.js)

13.2 Backend
     - Node.js + Express.js
     - SQLite database (better-sqlite3)
     - JWT authentication (jsonwebtoken)
     - IMAP sync (imapflow)
     - SMTP sending (nodemailer)
     - Email parsing (mailparser)
     - File uploads (multer)
     - CORS enabled
     - Real-time updates (Socket.IO)
     - 2FA support (speakeasy + qrcode)
     - Password hashing (bcryptjs)
     - Environment config (dotenv)

13.3 Server Architecture
     - REST API on port 3001
     - Static frontend served on port 3001
     - SQLite database (eclatrecon-mail.db)
     - Route modules:
       * auth.js         - Authentication & 2FA
       * emails.js       - Email CRUD & sync
       * campaigns.js    - Marketing campaigns
       * productivity.js - Calendar, tasks, notes, reminders
       * features.js     - Signatures, templates, scheduled, forwarding
       * security.js     - DLP, IP whitelist, GDPR
       * billing.js      - Plans & subscriptions
       * integrations.js - Third-party integrations
       * teams.js        - Team & shared mailboxes
       * analytics.js    - Email analytics
       * admin.js        - Admin panel
       * manage.js       - Account management
       * dataManagement.js - Backup & export
       * publicApi.js    - Public API endpoints
       * apify.js        - Apify lead scraper integration
